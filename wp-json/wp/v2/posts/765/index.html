After a leisurely start to the day, we decided to go see the Central Post Office. It was built between 1886 and 1891 when Vietnam was part of French Indochina.<\/p>\n\n\n\n

Apart from being a tourist attraction for its French architecture, it is still a working post office, so we bought some stamps!<\/p>\n\n\n\n

Across the street from the post office is Notre Dame Cathedral, also known as the Notre-Dame Cathedral Basilica of Saigon.<\/p>\n\n\n\n

This was built between 1863 and 1880 by French colonists and all the original building materials were imported from France.<\/p>\n\n\n\n

We arrived late on in the day so we couldn’t go inside. Also there was some building work going on so we weren’t quite sure if you could go in anyhow. However we managed to get a photo of the exterior before it got dark.<\/p>\n\n\n\n
